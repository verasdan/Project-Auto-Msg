## 💻 Sobre o projeto

Projeto Auto Msg - é uma ferramenta de automação feita em JavaScript para envio de mensagens ja definidas através da plataforma do Whatsapp Web.

---

## ⚙️ Funcionalidades

- [x] Envio de mensagens padronizadas através do Whatsapp Web.
